# PS Planungsbüro Schubiger AG — Recruiting Landing Page

Single-page recruiting funnel for PS Planungsbüro Schubiger AG (Uznach, CH). Candidates complete a 5-step application in under 2 minutes.

## Stack

- Pure HTML / CSS / vanilla JS — no build step
- Google Fonts (Jost + Fraunces)
- Brand color `#014898`

## Deploy

Static file — upload `index.html` to any host (Netlify, Vercel, GitHub Pages, S3, etc.) or open locally.

```bash
# local preview
python -m http.server 8000
```

## Structure

- Hero section — brand intro + CTA
- 5-step questionnaire (education → situation → priorities → benefits → contact)
- Success state on submit
- Data logged to `console` and `localStorage` (demo — wire to real backend before production)
